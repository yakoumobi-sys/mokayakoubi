HOW TO UPLOAD
════════════════════════════════════════════════════════════════════════

1. Go to: https://github.com/mokayakoubi/mokayakoubi (your new repo)

2. Click: "Upload files" button (middle of empty repo page)

3. Drag this entire folder into the upload area

4. Commit message: "Initial Next.js setup"

5. Click "Commit changes"

6. DONE! All files are on GitHub ✓

════════════════════════════════════════════════════════════════════════
IF UPLOAD FAILS
════════════════════════════════════════════════════════════════════════

Try uploading in groups:
  1. Upload all root files first (the .json, .js, .md files)
  2. Upload app/ folder contents
  3. Upload lib/ folder contents

GitHub will auto-create the folder structure.
